     A. Latar Belakang  
Pada tabel Post, secara default hanya terdapat tombol:  
• ✏ Edit 
Sedangkan tombol Delete tersedia di halaman edit.  
Agar lebih efisien, kita dapat:  
• Menambahkan tombol Delete langsung di tabel  
• Menambahkan tombol Replicate (Copy Data)  
• Membuat tombol custom (misalnya ubah status Publish/Unpublish)  
 
     B. Menambahkan Delete Action  
Buka file:  
• PostsTable.php

     E. Membuat Custom Action (Ubah Status Publish)  
Misalnya ingin mengubah status published langsung dari tabel tanpa masuk ke edit.  
 
  1. Tambahkan Custom Action  
• Action::make('status')  
    ->label('Status Change')  
 

=== PAGE 5 ===
 
  2. Tambahkan Form Input pada Action  
Gunakan schema:  
• ->schema ([ 
    Checkbox::make('published')  
        ->default (fn$record):bool =>$record ->published),  
]) 
 
Hasilnya s eperti berikut  
 
 
 

=== PAGE 6 ===
  3. Tambahkan Logic untuk Update Data  
• ->action (function  ($record , $data ) { 
                   $record ->update (['published'  => $data ['published' ]]); 
                }) 
 
 
  4. Tambahkan Icon  
• ->icon('heroicon -o-check -circle')  
 

=== PAGE 7 ===
 
 
     F. Contoh Lengkap Custom Action  
• Action ::make ('status' ) 
o ->label ('status change' ) 
o ->icon ('heroicon -o-check -circle' ) 
o ->schema ([ 
o Checkbox ::make ('published' ) 
o ->default (fn($record ): bool  => $record ->published),  
o ]) 
o ->action (function  ($record , $data ) { 
o $record ->update (['published'  => $data ['published' ]]); 
o }) 
o ]) 
o ->toolbarActions ([ 
o BulkActionGroup ::make ([ 
o DeleteBulkAction ::make (), 
o ]),

     H. Fitur Tambahan Action  
Filament juga menyediakan:  
• requiresConfirmation() → menambahkan konfirmasi  
• color() → ubah warna tombol  
• visible() → tampil berdasarkan kondisi  
• url() → redirect ke halaman lain  
• openUrlInNewTab() → buka di tab baru
